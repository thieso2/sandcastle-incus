Core Assertions

  • Tenant Incus projects are named sc-<tenant>.
  • Machine instances are named {project}-{machine}.
  • Machine private DNS names use {machine}.{project}.{tenant}.
  • Machine Caddy serves tenant-CA leaf certificates and proxies the configured app port.
  • Restricted users can mutate only tenants granted to their Incus certificate.
  • Public route authorization is based on tenant grants, not user-name ownership.
  • Cleanup removes disposable tenant projects, infrastructure projects, certificates, image aliases, route state, local DNS state, host entries, and local trust state for an explicit run ID.
